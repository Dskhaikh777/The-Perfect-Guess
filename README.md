# The Perfect Guess 🎯

A fun number guessing game built in Python!

## 🔹 Features
- Difficulty levels (Easy, Medium, Hard)
- Scoring system
- Time tracking and bonus
- High score tracking
- Replay option

## ▶️ How to Run

Make sure Python is installed.  
Install dependencies using:

```bash
pip install colorama
python perfect_guess.py
